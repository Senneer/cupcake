<h2>Как начать?</h2>
<ul>
  <li>Скачать проект</li>
  <li>Открыв терминал, перейти в папку с проектом</li>
  <li>Ввести команду "npm i"</li>
  <li>Для запуска проекта выполнить "npm start", после чего должен открыться браузер</li>
</ul>
